Silex templates

These templates are released by the authors and contributors as creative commons free templates. Anyone who pushes design and HTML files in this repository agrees to this license. We invite you to use these templates for your projects.

The templates can be edited with Silex, live web creation, the HTML editor and website builder for designers.

How to customize it for your website?

Download the zip file on github and unzip to a folder in your dropbox. Then open Silex in a browser and open the template file "editable.html".

Then insert your content, modify the design and use it for your website.

How to contribute

These templates are made without programming, just drag and drop, with Silex, live web creation. So feel free to fork this github repository, make changes and make a pull request.
