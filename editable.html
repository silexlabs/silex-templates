About Silex templates

These templates can be edited with Silex, live web creation, the HTML editor and website builder for designers.

How to customize it for your website?

Download the zip file and unzip to a folder in your dropbox. Then open Silex in a browser and with it, open the template file editable.html from the/templates/ folder.

Then insert your content, modify the design and use it as your website.

How to contribute

These templates are made without programming, just drag and drop, with Silex, live web creation. So feel free to fork this github repository, make changes and make a pull request.
