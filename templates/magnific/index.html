About this template


This is an HTML template to edit with Silex, the open source website builder. For you, designers and web designers to engage new customers.


This template is made without programming, just drag and drop and parameters. A little CSS too, but you will not need to touch it to change the design and content. 


This three pages template is very simple to edit without programming language, and then publish online. You own your data and Silex will always be free and open source. See Silex source code here.



Customize it for your website


1- Download the zip file, and unzip it to a folder in your dropbox. 

2- Then open Silex in a browser and then open the template file : "editable.html".

3- Add your content, drag, drop, modify the design and publish online.
